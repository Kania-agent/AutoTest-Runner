# AutoTest Runner

AI-powered test generation and runner dashboard built with MiMo V2.5.

## Features

- Clean dashboard with test suite overview
- Test case table with status, duration, and pass/fail badges
- CSS-only pie chart for code coverage
- Test generation via AI
- Run history and statistics

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (ES6+)
- No external dependencies or frameworks

## Usage

Open `index.html` in a browser. View test results, generate new tests, and monitor coverage metrics.

## AI Engine

Powered by **MiMo V2.5** — Nous Research's test generation model.
